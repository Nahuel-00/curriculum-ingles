VSC-GitHub

De lo que se trataba este proyecto era de poder sincronizar el VSC con el GitHub sin necesidad de el GitHub Desktop, Para esto debido de instalarnos una extensión, además de tener que instalar el Git el cual nos permitía subir los cambios que hemos realizado y realizar un commit el cual nos permitia volver a atras en caso de escribir un codigo erroneo.

HTML

Lo que he hecho a lo largo de este proyecto ha sido aprender una serie de instrucciones para poder configurar una pagina web, ya sea, como insertar una imagen, cambiar su tamaño y anchura, añadir enlaces a repositorios ya creados con antelación, etc.
